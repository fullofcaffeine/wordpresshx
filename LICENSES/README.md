# Licensing status

A final license has not been selected for the SDK, generic compiler work, examples, generated runtime support, generated output, or derived WordPress/Gutenberg/third-party contract metadata.

This directory is a placeholder for the reviewed license inventory and notices. Its presence is not a license grant. Do not publish public packages or release archives until ADR-020 and SDK-002 are accepted with qualified review.

The review must distinguish at least:

- maintained Haxe/Node/build source;
- generic PHP and browser compiler source;
- generated PHP/TS/JS and copied runtime support;
- examples, scaffolds, and templates;
- profiles and contracts derived from WordPress, Gutenberg, or third-party metadata;
- bundled Composer/npm/Haxelib dependencies and notices;
- final plugin/theme ZIPs, source archives, SBOM, and provenance.
